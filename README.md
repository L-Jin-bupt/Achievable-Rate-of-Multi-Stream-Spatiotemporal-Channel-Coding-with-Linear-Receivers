# Achievable-Rate-of-Multi-Stream-Spatiotemporal-Channel-Coding-with-Linear-Receivers
This simulation code package is used to reproduce the results of the following paper [1]:

[1] L. Jin, X. Xu, S. Han, X. Chi, P. Zhang, C. Yuen, "Achievable Rate of Multi-stream Spatiotemporal Channel Coding with Linear Receivers," IEEE Wireless Commun. Let., early access, Dec. 2025.  

https://ieeexplore.ieee.org/document/11276946
or
https://www.techrxiv.org/users/950460/articles/1319991

*********************************************************************************************************************************
The author in charge of this simulation code pacakge is: Liang Jin (email: jinliang@bupt.edu.cn).

Simulation environment: MATLAB R2021a.

*********************************************************************************************************************************
How to use this simulation code package?

Fig. 2 and Fig. 3 can be generated by directly running the "mainchangeD.m" file. 
Fig. 4 can be generated by directly running the "mainchangesnr.m" file. 
Fig. 5 can be generated by directly running the "mainlatency.m" file. 
Fig. 6 can be generated by directly running the "mainchannelhardening.m" file.
